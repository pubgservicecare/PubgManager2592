* /account/7
* /api/share/account/7

Example:

Instead of:

* /account/7

Use:

* /account/time-traveler-blood-raven-max-8x-sport-cars

Requirements:

* automatic slug generation
* clean readable URLs
* SEO-friendly structure
* preserve old links compatibility if needed
* avoid broken links
* support future Google indexing
